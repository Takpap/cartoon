# 时间线视图增强需求规格

## 概述

增强工作台的时间线视图组件，让用户能够更直观地查看和管理完整的场景序列，提供更好的视频制作工作流体验。

## 用户故事

### US-001: 查看完整场景序列
**作为** 视频创作者  
**我希望** 在时间线上看到所有场景的完整序列  
**以便** 我能够了解整个视频的结构和时长分布

**验收标准:**
- [ ] 时间线显示所有场景，按顺序排列
- [ ] 每个场景显示标题、时长、状态
- [ ] 场景之间的转场效果可见
- [ ] 总时长实时计算并显示

### US-002: 场景状态可视化
**作为** 视频创作者  
**我希望** 通过颜色和图标快速识别每个场景的生成状态  
**以便** 我能够了解哪些场景已完成、正在处理或待处理

**验收标准:**
- [ ] 视频完成的场景显示绿色
- [ ] 帧完成的场景显示黄色
- [ ] 生成中的场景显示蓝色并有加载动画
- [ ] 待处理的场景显示灰色
- [ ] 错误状态显示红色

### US-003: 时间线播放预览
**作为** 视频创作者  
**我希望** 能够在时间线上模拟播放，预览场景切换  
**以便** 我能够在生成视频前了解节奏和时长

**验收标准:**
- [ ] 播放/暂停按钮控制时间线播放
- [ ] 播放头随时间移动
- [ ] 可以点击时间线跳转到任意位置
- [ ] 显示当前时间和总时长

### US-004: 场景快速操作
**作为** 视频创作者  
**我希望** 能够从时间线直接选择和编辑场景  
**以便** 我能够快速调整场景内容

**验收标准:**
- [ ] 单击场景块选中该场景
- [ ] 双击场景块打开编辑对话框
- [ ] 选中的场景有明显的视觉反馈

### US-005: 展开详细视图
**作为** 视频创作者  
**我希望** 能够展开时间线查看更多场景细节  
**以便** 我能够在需要时获取更多信息

**验收标准:**
- [ ] 展开按钮切换详细视图
- [ ] 详细视图显示场景缩略图
- [ ] 详细视图显示镜头语言信息（景别、运镜）
- [ ] 详细视图显示转场设置

### US-006: 镜头语言信息展示
**作为** 视频创作者  
**我希望** 在时间线上看到每个场景的镜头语言设置  
**以便** 我能够确保视觉叙事的连贯性

**验收标准:**
- [ ] 显示景别类型（大远景、全景、中景等）
- [ ] 显示运镜方式（推、拉、摇、移等）
- [ ] 显示转场效果（硬切、淡变、叠化等）

## 技术要求

### 组件结构
- 组件路径: `app/components/workbench/TimelineView.vue`
- 使用 Vue 3 Composition API
- 使用 shadcn-vue 组件库
- 使用 lucide-vue-next 图标

### 数据接口
- 接收 `scenes: SceneData[]` 作为 props
- 接收 `selectedSceneId?: string` 标识当前选中场景
- 发出 `select-scene` 事件选择场景
- 发出 `edit-scene` 事件编辑场景

### 性能要求
- 支持 50+ 场景的流畅渲染
- 播放动画保持 60fps
- 响应式布局适配不同屏幕尺寸
